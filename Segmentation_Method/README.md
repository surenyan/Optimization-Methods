## Հատվածի կիսման եղանակ

Ենթադրենք $f(x)$ ֆունկցիան ունիմոդալ է $[a, b]$ հատվածի վրա, և անհրաժեշտ է գտնել $f(x)$-ի min-ի $x^*$ կետը՝ $\varepsilon > 0$ բացարձակ սխալանքի ճշտությամբ։

---

**_Քայլ 0:_** Ընտրենք $\delta \in (0, 2\varepsilon)$ մեծություն և նշանակենք $a_0 = a$, $b_0 = b$։ Որոշենք՝

$$x_1^{(0)} = \frac{a_0 + b_0 - \delta}{2}, \quad x_2^{(0)} = \frac{a_0 + b_0 + \delta}{2}$$

կետերը և հաշվենք $f(x)$-ի արժեքները այս կետերում, այսինքն՝ $f(x_1^{(0)})$-ը և $f(x_2^{(0)})$-ը։

---

**_Քայլ 1:_** Որոշենք որոնման նոր $[a_1, b_1]$ միջակայքը՝ համեմատելով $f(x_1^{(0)})$ և $f(x_2^{(0)})$ արժեքները հետևյալ կերպ՝

- ա) եթե $f(x_1^{(0)}) \leq f(x_2^{(0)})$, ապա ընդունենք $a_1 = a_0,\ b_1 = x_2^{(0)}$
- բ) եթե $f(x_1^{(0)}) > f(x_2^{(0)})$, ապա ընդունենք $a_1 = x_1^{(0)},\ b_1 = b_0$

$[a_1, b_1]$ նոր միջակայքի վրա դիտարկենք հետևյալ կետերը՝

$$x_1^{(1)} = \frac{a_1 + b_1 - \delta}{2}, \quad x_2^{(1)} = \frac{a_1 + b_1 + \delta}{2}$$

և հաշվարկենք $f(x_1^{(1)})$-ը և $f(x_2^{(1)})$-ը։

---

**_Քայլ 2:_** Այնուհետև՝

- ա) եթե $f(x_1^{(1)}) \leq f(x_2^{(1)})$, ապա ընդունենք $a_2 = a_0,\ b_2 = x_2^{(1)}$
- բ) եթե $f(x_1^{(1)}) > f(x_2^{(1)})$, ապա ընդունենք $a_2 = x_1^{(1)},\ b_2 = b_0$

---

**_Քայլ k ($k \geq 1$):_** Որոշենք որոնման նոր $[a_k, b_k]$ միջակայքը և՝

$$x_1^{(k-1)} = \frac{a_{k-1} + b_{k-1} - \delta}{2}, \quad x_2^{(k-1)} = \frac{a_{k-1} + b_{k-1} + \delta}{2}$$

կետերը, ինչպես նաև $f(x)$-ի արժեքները այդ կետերում, այսինքն՝ $f(x_1^{(k-1)})$-ը և $f(x_2^{(k-1)})$-ը։ Որոնումն ավարտվում է, երբ՝

$|b_n - a_n| = \varepsilon$, որտեղ $n$-ը հնչ-nnp թիվ է։ Որպես min-ի կետի մոտավորություն վերցվում է $x_m = 0.5(a_n + b_n)$-ը, իսկ որպես $f(x)$ ֆունկցիայի min արժեք՝ $f(x_m)$ մեծությունը։

Այդ դեպքում անհրաժեշտ է, որ ստացված $x_m$ մոտավորությունը մոտ լինի $x^*$ ճշրիտ լուծմանը և $|x_m - x^*| \leq \varepsilon$, որտեղ $\varepsilon$-ը նախապես տրված թիվ է։ Այդ դեպքում $x^*$ կետի որոշման առավելագույն $\varepsilon_n$ սխալանքը՝

$$\varepsilon_n = 0.5 \cdot (b_n - a_n)$$
